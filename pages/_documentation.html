MQTT Client library for C

Documentation on how to use the MQTT client along with examples and additional information on important concepts can be found here: MQTT C Client Documentation, and Asynchronous MQTT C Client Documentation.

IBM Redbook on MQTT

Provides an introduction to the MQTT protocol and uses a scenario-based approach to demonstrate its capabilities. It is intended as a quick guide to getting started, including concepts of enterprise scale messaging using an MQTT enabled server. MQ Telemetry Transport Redbook.
